#  FaceAttend AI
**Sistem Absensi Cerdas Berbasis Face Recognition & Multi-Layer Liveness Detection**

![Python](https://img.shields.io/badge/Python-3.12-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-0.109-009688)
![License](https://img.shields.io/badge/License-MIT-green)

## 🚀 Fitur Utama
- ✅ **Face Recognition** akurat menggunakan InsightFace / Face-API.js
- 🛡️ **Anti-Spoofing** dengan Liveness Detection (tidak bisa pakai foto/video)
- ⚙️ **Admin Panel** untuk atur jam masuk & toleransi keterlambatan secara real-time
- 📊 **Dashboard Interaktif** dengan statistik harian & export Excel
- 🗑️ **Auto-Clean Database** (hapus user otomatis menghapus riwayat absensi)
- 📱 **Responsive UI** modern dengan glassmorphism theme
